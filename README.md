## Installation

```
# npm install ctgen -g
```

## Use

```
$ ctgen install -d namedb
```
```
$ ctgen --help
```
## Use auth

```
$ ctgen install -d ctbook --verbose -u ctbook -p "password"
```


## Test

```
$ npm install
```
```
$ npm test
```
